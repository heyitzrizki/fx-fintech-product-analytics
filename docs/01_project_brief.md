# FX Fintech Product Analytics Case Study

## Project Overview

This project is a synthetic product analytics case study inspired by a foreign exchange fintech platform. The objective is to analyze user onboarding, activation, repeat transaction behavior, feature adoption, FX market behavior, and customer value using SQL-based analytical marts and dashboard-ready outputs.

The analysis uses a relational SQLite database containing user profiles, product events, exchange transactions, FX rates, marketing spend, support tickets, and modeling-ready monthly user snapshots.

## Business Context

FX fintech platforms need to understand not only how many users sign up, but also whether users complete onboarding, make their first exchange, repeat transactions, and adopt product features such as rate alerts or target-rate orders.

This project focuses on answering product and growth questions such as:

- Which acquisition channels generate the strongest activation quality?
- Which user behaviors are associated with repeat exchange activity?
- Do target-rate and rate-alert features relate to stronger retention?
- How does FX market volatility affect transaction behavior and success rate?
- Which customer value segments show deeper product engagement?

## Data Scope

The dataset covers synthetic user and transaction activity across 2024–2025.

Main tables:

- users
- events
- transactions
- fx_rates_hourly
- marketing_spend
- support_tickets
- modeling_user_month_snapshots

## Important Note

This project uses a synthetic dataset created for portfolio and analytical demonstration purposes. The results should not be interpreted as actual Switchwon company performance.
